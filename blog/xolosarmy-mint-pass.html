¿Qué es el Mint Pass en XoloLegend?

Un Mint Pass no es un NFT “bonito” para presumir: es un permiso on-chain que autoriza mintear (crear) 1 NFT dentro del ecosistema XoloLegend. Piensa en él como una llave digital: soberana, verificable y ligada a un Offer ID.

Verificable on-chain 1 permiso = 1 mint Integración con Tonalli Wallet Flujo “buy → verify → redeem”

Formato clave

En XoloLegend, cada listing se ancla a un Offer ID con formato txid:vout. Ese identificador permite verificar que el outpoint existe y que el offer es real.

Offer ID = <txid>:<vout> Ejemplo: 52b9ce...43ef2c3:1
Tip: txid es un hash de 64 chars hex. vout es un índice (0,1,2…).

Concepto

El Mint Pass es un derecho de creación. No “representa” arte: autoriza el acto de mintear. Cuando lo usas, su propósito se cumple (en términos prácticos, se considera consumido o “usado”).
